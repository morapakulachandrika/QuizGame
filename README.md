# QuizGame

# 🎯 Quiz Game (HTML, CSS, JavaScript)

An interactive and responsive **Quiz Web Application** built using **HTML, CSS, and JavaScript**, featuring real-time answer feedback and dynamic scoring.

---

## 🚀 Features

✨ Multiple Quiz Categories
⚡ Instant Answer Feedback (correct answer shown immediately)
🎯 Dynamic Score Calculation
🎨 Clean and Interactive User Interface
📱 Responsive Design (works on all devices)

---

## 🖥️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling
* **JavaScript (Vanilla JS)** – Functionality
